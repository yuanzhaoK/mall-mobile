
## 项目简介

这是一个基于Flutter开发的移动端家装商城应用，专注于为用户提供优质的家装产品和服务。应用采用现代化的UI设计和模块化的代码架构，提供直观的用户体验。

## 项目结构

```
lib/
├── main.dart                    # 应用入口文件
├── constants/                   # 常量定义
│   ├── app_colors.dart         # 颜色常量
│   └── app_strings.dart        # 字符串常量
├── models/                      # 数据模型
│   ├── product.dart            # 产品相关模型
│   └── data_source.dart        # 数据源
├── pages/                       # 页面文件
│   ├── home_page.dart          # 首页
│   ├── mall_page.dart          # 商场页面
│   └── profile_page.dart       # 个人中心
└── widgets/                     # 可复用组件
    ├── feature_button.dart     # 功能按钮组件
    ├── selection_row.dart      # 选择行组件
    ├── recommendation_card.dart # 推荐卡片组件
    ├── product_card.dart       # 产品卡片组件
    └── menu_item.dart          # 菜单项组件
```

## 功能特点

### 🏠 首页
- **品牌展示**：装库科技品牌信息展示
- **功能快捷入口**：全部礼包、装修灵感等快捷功能
- **个性化选择**：支持户型选择和礼包定制
- **精品推荐**：展示多种风格的家装套餐
  - 现代简约套装
  - 北欧风格套装
  - 中式传统套装

### 🛍️ 商场
- **产品展示**：网格布局展示各类家具产品
- **分类浏览**：沙发、床、餐桌、衣柜等多种家具类型
- **价格展示**：清晰的价格标签
- **购物车功能**：支持一键加入购物车

### 👤 个人中心
- **用户信息**：个人资料和VIP会员状态
- **订单管理**：我的订单、收藏等功能
- **地址管理**：收货地址管理
- **客服支持**：在线客服和帮助中心

## 技术栈

- **框架**：Flutter 3.32.4
- **开发语言**：Dart
- **UI设计**：Material Design 3
- **状态管理**：StatefulWidget
- **架构模式**：模块化分层架构
- **平台支持**：iOS、Android、Web

## 设计特色

- **黄色主题**：采用温暖的黄色调，营造温馨的家居氛围
- **渐变背景**：首页使用渐变背景，视觉效果丰富
- **圆角设计**：大量使用圆角元素，界面更加柔和
- **阴影效果**：适当的阴影增加界面层次感
- **响应式交互**：点击反馈和Toast提示，提升用户体验
- **模块化组件**：可复用的UI组件，便于维护和扩展

## 代码架构

### 📁 常量管理
- **AppColors**: 统一管理应用颜色主题
- **AppStrings**: 集中管理所有文本内容，便于国际化

### 📊 数据层
- **Product**: 产品数据模型
- **PackageModel**: 礼包数据模型  
- **SuiteModel**: 套装数据模型
- **DataSource**: 静态数据源，提供应用所需数据

### 🎨 UI组件
- **FeatureButton**: 可复用的功能按钮
- **SelectionRow**: 选择行组件
- **RecommendationCard**: 推荐卡片
- **ProductCard**: 产品展示卡片
- **MenuItem**: 菜单项组件

### 📱 页面结构
- **HomePage**: 首页，包含品牌展示、功能入口、推荐内容
- **MallPage**: 商场页面，展示产品列表
- **ProfilePage**: 个人中心，用户信息和功能菜单

## 安装运行

### 环境要求
- Flutter SDK 3.0+
- Dart SDK 2.17+
- Android Studio / VS Code
- Chrome浏览器（Web预览）

### 运行步骤

1. **获取项目**
   ```bash
   # 如果从git克隆
   git clone [项目地址]
   cd flutter_home_mall
   ```

2. **安装依赖**
   ```bash
   flutter pub get
   ```

3. **运行项目**
   
   Web版本：
   ```bash
   flutter run -d chrome
   ```
   
   Android版本：
   ```bash
   flutter run -d android
   ```
   
   iOS版本：
   ```bash
   flutter run -d ios
   ```

## 核心功能说明

### 礼包选择功能
用户可以通过底部弹窗选择不同的装修礼包：
- 现代简约礼包（￥99,999）
- 北欧风格礼包（￥89,999）  
- 中式传统礼包（￥129,999）

### 商品推荐
首页展示三种不同风格的套装推荐，每个套装包含：
- 套装介绍和价格信息
- 好物推荐按钮
- 立体呈现按钮（3D效果预览）

### 购物体验
商场页面提供8种不同类型的家具产品，支持：
- 网格布局浏览
- 详细价格和描述显示
- 一键加入购物车
- Toast反馈提示

## 开发说明

### 代码特点

- **模块化设计**：页面、组件、数据、常量分离，便于维护
- **可复用组件**：提取通用UI组件，减少代码重复
- **统一样式管理**：通过常量文件统一管理颜色和文字
- **响应式布局**：适配不同屏幕尺寸
- **Material Design**：遵循Google设计规范
- **代码注释**：关键代码都有详细注释

### 扩展指南

1. **添加新页面**：在`pages/`目录下创建新文件
2. **新增组件**：在`widgets/`目录下添加可复用组件
3. **数据管理**：在`models/`目录下定义数据模型
4. **样式修改**：通过`constants/`目录统一修改主题

## 性能优化

- 使用`const`构造函数减少重建
- 合理使用`SingleChildScrollView`提升滚动性能
- 图标使用系统Icons，减少资源占用
- 模块化导入，减少不必要的依赖
